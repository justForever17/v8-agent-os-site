Core runtimes

Each runtime owns one job well.

Use the right runtime instead of forcing every task through chat alone.

CHAT RUNTIME

Plan, answer, delegate, and keep the run ledger coherent.

MEMORY RUNTIME

Preferences, knowledge, artifacts, graph retrieval, and recall.

AUTOMATION RUNTIME

Cron, hooks, and heartbeat wake-ups that keep work moving.

PLUGIN HOST RUNTIME

OpenClaw tools, IM bridges, and external channels on one surface.

COMPUTER USE RUNTIME

Desktop actions and higher-friction execution under guardrails.

RPA RUNTIME

Structured automation for repeated UI and workflow tasks.

NETWORK SUPERVISOR RUNTIME

LAN discovery, WAN bootstrap, directed wake, signed envelopes, and multi-peer coordination.

Install

Install V8 Agent OS in one command.

Pick your platform, copy the command, and start the latest engine and admin stack.

Windows
powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/justForever17/v8-agent-os-engine/main/scripts/install/bootstrap.ps1 | iex"
Open script
Linux
curl -fsSL https://raw.githubusercontent.com/justForever17/v8-agent-os-engine/main/scripts/install/bootstrap.sh | bash
Open script
macOS
curl -fsSL https://raw.githubusercontent.com/justForever17/v8-agent-os-engine/main/scripts/install/bootstrap.sh | bash
Open script
What you get

The script pulls the latest engine and admin repos, installs dependencies, creates a local admin env file, and starts both services.
