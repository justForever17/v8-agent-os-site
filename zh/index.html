Desktop Preview // Supervisor First, Runtime Grounded

不止于对话。
这是面向长期任务的桌面 Agent OS。

V8 把 Supervisor 的判断放在任务前台，把执行建立在 runtime 状态、证据、产物和明确边界上。
桌面端是主要工作区；编程、调研、多媒体、桌面操作和 RPA 各自承担专门执行，配对后的 Phone 用于远程续接任务与处理受治理的确认。

安装桌面预览版 查阅核心源码
[01] MEMORY_GRAPH

基于证据的记忆与图谱审阅

记忆保存的是证据、偏好、图关系和产物，而不是一条可以替代当前指令的命令流。维护任务与 Admin 图谱帮助用户审阅、修改并追踪本轮究竟召回了什么。

[02] COST_ROUTING

按角色配置模型与故障回流

Supervisor、子 Agent、多媒体、向量和辅助任务可以使用不同模型与供应商。健康检查和 failover 会明确暴露降级路径，而不是静默掩盖。

[03] REMOTE_OPS

配对 Phone 的远程任务续接

桌面端始终是主要工作区。配对后的 Phone 可以继续任务、回答阻塞问题、发送语音或文件、查看产物，并在离开电脑时处理受治理的副作用确认。

[04] PLUGIN_MANAGER

受治理的插件管理器

从签名目录安装经过审核的 CLI、MCP、Skill 与 UI 组件，现已包含高德官方 CLI。@插件 是强提示；Supervisor 也可为当前任务授权最小就绪组件，并只向直接子 Agent 传递所需子集。

[05] ZERO_TRUST_OPS

受治理的副作用与权限边界

工作区边界、类型化 CLI 动作、审批、授权摘要与逐次调用校验共同约束特权操作。被阻断或降级时，系统返回原因和恢复入口，而不是伪造成功。

[06] DYNAMIC_EXPOSURE

按任务预筛的 Skill 与 MCP

通用 Skill 和 MCP 只按当前任务进入候选面，避免把完整工具目录塞给模型。插件拥有的组件保持隔离，只有有效 task grant 才会投影。

[07] UI_TO_RPA

从已验证桌面操作沉淀 RPA

Computer Use 负责观察并完成一次性桌面任务；只有作用域、环境、时序和验证证据明确后，重复步骤才会被提升为可复用 RPA。

[08] AUTO_RUNTIME

可恢复的 Runtime Episode 与 Handoff

队列、租约、类型化 handoff、产物和恢复状态让复杂执行在暂停与重启后仍可检查。最终仍由 Supervisor 验收回流证据，再决定是否完成。

[09] SESSION_TAKEOVER

安全的跨会话任务接管

新任务可以引用旧任务的目标、确认决定、Spec 状态、产物、失败和阻塞，但不会继承旧工作区、权限、checkpoint 或 runtime episode。

待补当前截图 跨会话任务接管 后续替换为 Web 与 Phone 的会话引用流程。
[10] SPEC_DELEGATION

Spec 模式与委派合同

需求、设计和任务阶段保留稳定 ID 与审批。子 Agent 回流继续携带 lineage、精确写集、自检、产物和父级验收，不再被压成一句模糊摘要。

待补当前截图 Spec 与委派合同 后续替换为阶段审批与 handoff 证据。
[11] CREATIVE_MEDIA

带产物 QA 的多媒体创作

六个稳定 facade 覆盖能力、计划、素材、作业、编辑和质量。recipe、provider lock、样片确认、artifact proof 与 QA 始终留在交付链上。

待补当前截图 多媒体创作流程 后续替换为 recipe、产物与 QA 页面。
[12] DESKTOP_COMPANION

由 Shell 管理的桌宠伴随器

桌宠通过本机鉴权控制通道跟随当前任务，在桌面应用内打开设置，保存后即时应用外观，并在强制兜底前优先执行自身的优雅关闭动画。

待补当前截图 桌宠任务同步与生命周期 后续替换为任务跟随与托管状态页面。
Deployment // Desktop Preview

一个桌面预览入口。

当前公开渠道是 unsigned 桌面预览版，不代表已签名 stable 或自动更新承诺。它把界面壳、治理层和执行运行时收进桌面体验；Phone 仍是配对后的远程交互面。

Windows (PowerShell)

powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/justForever17/v8-agent-os/main/bootstrap.ps1 | iex"

Linux / macOS (Bash)

curl -fsSL https://raw.githubusercontent.com/justForever17/v8-agent-os/main/bootstrap.sh | bash
