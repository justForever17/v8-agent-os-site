初始化 Engine 查阅核心源码
[01] MEMORY_GRAPH

分层记忆治理与可视化干预

真正的记忆不只是丢个 CLAUDE.md 文件导致常常失忆盲猜。V8 内置独立 Memory Agent 进行后台清洗，深度结合 RAG 与 Supervisor。人类更可在前端拓扑图，像外科医生一样精准查阅和修改每一丝知识脉络。

[02] COST_ROUTING

精算级模型路由

系统原生拦截并向下路由至免费的本地小模型接管杂活与摘要，将昂贵的前沿算力用在核心推理锋刃上。

[03] REMOTE_OPS

高位跨端的移动治理台

Agent 系统不应限于 PC 命令行。通过 Web 远程伴生端，你可以在咖啡厅用手机随地查看后台并发任务、监控多智能体对话，还能轻触完成高风险的 安全指令越权审批。

[04] ECOSYSTEM_HIJACK

对 OpenClaw 生态的逆向接管

抛弃重启炉灶。通过黑科技 v8-bridge，V8 原生劫持并全量兼容 OpenClaw 庞大的开源插件社区，将其化为你的前遣火力，并牢牢套上 V8 的安全约束。

[05] ZERO_TRUST_OPS

白盒安全隔离审批

盲猜瞎挂未知 Skills 等于向黑客开门。遇到极危指令，必须通过前端人机审核。哪怕藏着的 rm -rf 也休想逃过你的凝视。

[06] DYNAMIC_EXPOSURE

MCP与SKILLS归一化预筛

通过顶层 Reranker 将数百个零散的 MCP 工具与原生 SKILLS 统一步调进行“归一预筛”。不溢出 Token 显存，只留给模型当下最精准的利刃。

[07] UI_TO_RPA

脆弱动作变现为 RPA

将不可靠的高时延 Computer Use 全平台（Win/Mac/Linux）动作阵列捕获，提炼转化成能无限调用的企业级 RPA 资产流水线。

[08] AUTO_RUNTIME

低耦合隔离：真正的自治运转舱

告别关闭浏览器就崩盘的虚假独立。结构上强隔离的 Control Plane (Admin 9528) 与 Execution Core (Engine 9530) 让数日的复杂跑图能够在后台稳健存活、断点唤醒，以及静候多端人工指令。

Deployment // Engine Bootstrap

一条指令，构建设施。

快速拉起双平面组件。

Windows (PowerShell)

powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/justForever17/v8-agent-os/main/bootstrap.ps1 | iex"

Linux / macOS (Bash)

curl -fsSL https://raw.githubusercontent.com/justForever17/v8-agent-os/main/bootstrap.sh | bash
