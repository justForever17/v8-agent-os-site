Runtime-first Agent OS

把聊天、记忆、自动化和真实任务收进同一条控制平面。

V8 Agent OS 面向可恢复的长期任务，把执行、记忆、自动化、插件宿主工具和多节点协作放在同一条 runtime 主链上。

  • 以 runtime 为核心的执行链
  • 可恢复的长任务
  • Skills、MCP 与插件宿主统一协作

核心运行时

每个 runtime 负责一类真正的工作。

不要把所有事情都塞回聊天。让正确的 runtime 处理正确的问题。

CHAT RUNTIME

规划、回答、委派，并保持 run ledger 一致。

MEMORY RUNTIME

管理偏好、知识、产物、图谱检索与 recall。

AUTOMATION RUNTIME

提供 cron、hooks 与 Supervisor 心跳唤醒。

PLUGIN HOST RUNTIME

统一 OpenClaw 工具、IM 桥接与外部通道。

COMPUTER USE RUNTIME

在护栏之下执行桌面动作与高摩擦任务。

RPA RUNTIME

承担结构化的界面自动化与重复流程任务。

NETWORK SUPERVISOR RUNTIME

支持局域网发现、广域网接入、定向唤醒、签名 envelope 与多 peer 协作。

安装

一条命令启动 V8 Agent OS。

按平台复制命令，就能拉取最新版的 engine 与 admin 并直接启动。

Windows
powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/justForever17/v8-agent-os-engine/main/scripts/install/bootstrap.ps1 | iex"
打开脚本
Linux
curl -fsSL https://raw.githubusercontent.com/justForever17/v8-agent-os-engine/main/scripts/install/bootstrap.sh | bash
打开脚本
macOS
curl -fsSL https://raw.githubusercontent.com/justForever17/v8-agent-os-engine/main/scripts/install/bootstrap.sh | bash
打开脚本
你会得到什么

脚本会拉取最新的 engine 和 admin 仓库、安装依赖、生成本地 admin env 文件，并启动两端服务。
